## Tools Required to Run

1. **.NET SDK (v6 or later)**
   - Needed to run the app.
   - Download: https://dotnet.microsoft.com/en-us/download

2. **Google Chrome**
   - Required for browser automation.

3. **ChromeDriver**
   - Already included in the folder.
   - Must match your Chrome version.

4. **Telegram.Bot & Selenium WebDriver**
   - Already included in the published files.
